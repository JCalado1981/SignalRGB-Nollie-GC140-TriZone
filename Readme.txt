# SignalRGB Nollie GC140 TriZone

Bring independent RGB lighting zones to your Nollie GC140 fans.

This project transforms the original 34 LED Nollie GC140 into three independent SignalRGB components:

• Center (10 LEDs)
• Left Side (12 LEDs)
• Right Side (12 LEDs)

Designed for advanced lighting effects similar to Lian Li UNI FAN devices while keeping full compatibility with the original Nollie hardware.

## Features

✔ Independent RGB Zones

✔ Compatible with SignalRGB

✔ Compatible with Nollie 8 Controller

✔ Daisy Chain Support

✔ Works with multiple fans on the same channel

## Components

GC140 Center

GC140 Left

GC140 Right

## Installation

Copy the JSON files to:

SignalRGB\\Components

Restart SignalRGB.

Add the three components to the Nollie controller.

Enjoy!
